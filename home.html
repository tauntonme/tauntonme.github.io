Signals for West Buckland Miniature Railway

The purpose of the signalling system is to promote safer running of the railway.


Taunton Model Engineers are working on designing and building a signalling system. These pages will contain every detail of the design and build as it progresses, 'Open Sourcing' ensures no one person achieves indispensibility, and that as time goes by, others will have the information needed to carry on, solving all the usual problems along the way.


These pages will be transferred to our own "West Buck Signalling" web site once set up, have been transferred to our "West Buck Signalling" web site on github.com enabling updates from any member in the Signalling Team.

TME Signalling Team members Please Note - You are encouraged to add to and modify the content saved here.

Ask J.F about joining the team of 'collaborators'. It's not too difficult, it goes like this :

  1. J.F. uses your email address to 'invite' you to join in GitHub collaboration
  2. GitHub email you an invite to sign up
  3. Should you be agreeable and sign up, you will gain read/write acces to the file repository!

For the basics of using 'GitHub', see loads of help and tutorials on Youtube.


All design files of ANY type, and absolutely everything to do with the signalling system will be found here.
